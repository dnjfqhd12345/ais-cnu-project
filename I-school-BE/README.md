# 🍦아이스쿠루무
🍦일본 대학생들을 위한 대학 생활 필수앱, 아이스쿠루무🍦

## Commit Convention
| Commit Type | Description                                |
|-------------|--------------------------------------------|
| feat        | 새로운 기능 추가                                  |
| fix         | 버그 수정                                      |
| docs        | 문서 수정                                      |
| style       | 코드 스타일 변경(코드 형식, 세미콜록 누락 등 기능을 수정하지 않은 경우) |
| refactor    | 코드 리팩토링                                    |
| test        | 테스트 코드 추가 및 수정                             |
| chore       | 패키지 매니저 수정                                 |
| rename      | 파일 또는 폴더를 삭제만 한 경우                         
| remove      | 파일명 또는 폴더명을 수정만 한 경우                       |
| build       | 빌드 관련 파일 수정
